# Avium Time Counter

Live IC/OOC time converter for the Avium nation RP universe.

## Features

- **Live clock** — displays the current in-character date and time, updating every IC second
- **Bidirectional calculator** — convert between OOC and IC dates with GMT offset support
- **Epoch system** — supports timeskips and rate changes via a code-editable `EPOCHS` array

## Usage

Open `index.html` in a browser. No build step, no dependencies.

## Deploy

Push to `main` — GitHub Pages serves `index.html` directly.

Live at [auroruse.github.io/avium-time-counter](https://auroruse.github.io/avium-time-counter/).
